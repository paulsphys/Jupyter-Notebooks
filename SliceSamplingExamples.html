Slice Sampling

Slice sampling is a MCMC method for sampling a probability distribution. It addresses one of the drawbacks of Metropolis-Hastings by adapting the step size according to the target distribution.

Radford M. Neal. Slice sampling. Ann. Statist. 31 (3) 705 - 767, June 2003

The following paper compares metropolis-hastings vs slice sampling

Denny Dittmar, Slice Sampling, TU Darmstadt

We illustrate slice sampling in the following cells

def SliceSamplerStep(x,w,p):
    u = np.random.uniform(0,p(x))
    upr = np.random.random()
    #Defines an interval
    xmin = x - upr*w
    xmax = xmin + w
    #"Steps out" accordingly
    while u < p(xmin):
        xmin = xmin - w
    while u < p(xmax):
        xmax = xmax + w
    while True:
        upr = np.random.random()
        xpr = xmin + upr*(xmax - xmin)
        if u < p(xpr):
            return xpr
        if xpr < x:
            xmin = xpr
        else:
            xmax = xpr

def LogSliceSamplerStep(x,w,p):
    logu = np.log(p(x)) - np.random.exponential()
    upr = np.random.random()
    xmin = x - upr*w
    xmax = xmin + w
    while logu < np.log(p(xmin)):
        xmin = xmin - w
    while logu < np.log(p(xmax)):
        xmax = xmax + w
    while True:
        upr = np.random.random()
        xpr = xmin + upr*(xmax - xmin)
        if logu < np.log(p(xpr)):
            return xpr
        if xpr < x:
            xmin = xpr
        else:
            xmax = xpr

Now let us try Elliptic Slice Sampling. Elliptic Slice Sampling was originally proposed for target distributions like $\pi(x) \sim \mathcal{N}(\mu,\Sigma)L(x)$. It works by randomly sampling $\nu \sim \mathcal{N}(\mu,\Sigma)$ and then uniformly sampling from the ellipse defined by $(x - \mu)cos(\theta) + (\nu - \mu)sin(\theta)$. The union of ellipses for all $\nu$ covers the whole slice thus their is no need to do any "stepping out" as we do for naive slice sampling. We illustrate this union in the final cell.

[1] Murray, I, Adams, RP & MacKay, DJC 2010, Elliptical slice sampling in (AISTATS)

[2] Robert Nishihara, Iain Murray, and Ryan P. Adams. 2014. Parallel MCMC with generalized elliptical slice sampling. J. Mach. Learn. Res. 15, 1 (January 2014), 20872112

The second paper extends the method to any arbitrary distribution.

def EllipticSliceSampler(x,mu,Sig,L):
    nu = np.random.normal(mu,Sig)
    u = np.random.random()
    logy = np.log(L(x)) + np.log(u)
    theta = np.random.uniform(0,2*np.pi)
    thetamin = theta - 2*np.pi
    thetamax = theta
    loopcounter = 0 
    while True:
        xpr = (x - mu)*np.cos(theta) + (nu - mu)*np.sin(theta) + mu
        if logy < np.log(L(xpr)):
            #print(loopcounter)
            return xpr
        else:
            if theta < 0:
                thetamin = theta
            else:
                thetamax = theta
            theta = np.random.uniform(thetamin,thetamax)
        loopcounter += 1
